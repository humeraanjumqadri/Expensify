# 💰 Expensify - Expense Tracker

A responsive expense tracker web app built using HTML, CSS, and JavaScript.

## Features
- Add, edit, delete transactions
- Income vs Expense tracking
- Dark mode toggle
- Search transactions
- Data saved in LocalStorage
- Chart visualization (Chart.js)

## Tech Used
- HTML
- CSS
- JavaScript
- Chart.js
